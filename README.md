# WinFormsCargoApp

**Приложение для управления грузоперевозками** на платформе Windows Forms с использованием .NET.

## Описание проекта

WinFormsCargoApp - это десктопное приложение для автоматизации работы транспортной компании. Программа позволяет управлять клиентами, тарифами и заказами с системой расчета стоимости и скидок.

##  Основные возможности

###  Управление данными
- **Клиенты**: добавление клиентов двух типов (Обычный/Постоянный)
- **Тарифы**: система тарифов с весовыми категориями
- **Заказы**: создание, редактирование, удаление заказов

###  Расчет стоимости
- Автоматический расчет стоимости заказа
- Система скидок (10% для постоянных клиентов)
- Динамический пересчет при изменении параметров

###  Работа с файлами
- Сохранение данных в JSON-формат
- Загрузка данных из JSON-файлов
- Встроенные тестовые данные для демонстрации

###  Пользовательский интерфейс
- Интуитивно понятный интерфейс Windows Forms
- Сортировка заказов по всем колонкам
- Визуальные индикаторы сортировки
- Подтверждение критических действий

##  Структура проекта

### Основные классы:

| Класс | Назначение |
|-------|------------|
| `CargoCompany` | Бизнес-логика приложения |
| `MainForm` | Главная форма с таблицей заказов |
| `OrderForm` | Форма создания/редактирования заказов |
| `FileService` | Сервис для работы с файлами (JSON) |
| `DataModel` | Модель данных для сериализации |

### Модели данных:
- **`Order`** - заказ клиента
- **`Client`** - клиент компании (с типом и историей заказов)
- **`Tariff`** - тариф с весовыми ограничениями
